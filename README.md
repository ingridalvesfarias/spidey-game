# 🕸️ Spider-Man vs Green Goblin - Battle Game

Este é um projeto de jogo de luta 2D desenvolvido com **HTML5 Canvas** e **JavaScript puro**. O projeto apresenta um confronto clássico entre o Homem-Aranha e o Duende Verde, com foco em uma interface **Cyber-Minimalista** e responsividade total para dispositivos móveis.

## 🚀 Tecnologias Utilizadas

* **HTML5 & CSS3**: Estrutura e estilização com variáveis modernas e animações dinâmicas.
* **JavaScript (ES6+)**: Lógica do jogo, detecção de colisão, inteligência artificial básica e manipulação do DOM.
* **Canvas API**: Renderização gráfica de alta performance para o combate.

## 🎮 Funcionalidades

* **Sistema de Combate**: Movimentação lateral, pulo e ataque de teia.
* **Interface Gamer**: HUD com barras de vida neon, placar em tempo real e visual inspirado em jogos de luta clássicos.
* **Totalmente Responsivo**: Controles touch integrados que aparecem automaticamente apenas em dispositivos móveis.
* **Mensagens de Vitória**: Feedback visual centralizado ao finalizar o round.

## 🕹️ Como Jogar

### No Computador (Desktop)
* **[A] / [D]**: Mover para esquerda ou direita.
* **[W]**: Pular.
* **[F]**: Atacar com teia.

### No Celular (Mobile)
* Utilize os **botões touch** circulares na parte inferior da tela para controlar o herói.

### Jogue Agora!
Acesse: https://spidey-game.vercel.app/

## 🎨 Design Aesthetic

O projeto segue uma linha de design **Cyberpunk/Dark Mode**:
* Background preto profundo para maior contraste.
* Acentos em **Verde Neon** e **Vermelho** para destacar os personagens e a UI.
* Efeitos de **Glow** (brilho) nas barras de vida e sombras projetadas no título.
